# Contact Us

Under Development

We're evaluating secure communication methods that align with our privacy principles. Options being considered include:

Once we've implemented proper verification mechanisms to prevent impersonation, this page will be updated with official contact methods.

Security Disclosure

If you've discovered a security vulnerability in content on this site (dangerous commands, insecure configurations, privacy issues), please note it for now. We'll establish a proper responsible disclosure process soon. In the meantime, you can use GitHub Issues on our repository once it's public.
